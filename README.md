# Freezer Monitor

This project monitors the temperature of a freezer and handles alerts based on temperature readings.

## Installation

1. Clone the repository: `git clone https://github.com/ViCor44/freezer-monitor.git`
2. Install dependencies.
3. Configure the .env file.
4. Run the application.

## Usage

1. Start the server.
2. Access the application at http://your-server-address.
